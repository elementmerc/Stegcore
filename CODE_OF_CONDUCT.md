# Code of Conduct

Stegcore's issue tracker, discussions, and every other space tied to this project are for thoughtful, respectful collaboration. Everyone is welcome regardless of experience, background, identity, or circumstance.

## Expected behaviour

- Be kind. Assume good faith in others, especially across time zones and language gaps.
- Be precise. When you report a bug or flag a concern, give enough detail for someone else to reproduce and act on it.
- Be on topic. This project is a steganography and analysis tool. Keep discussions focused on the work.
- Respect people's time. Search first before asking. Read error messages fully. Provide versions, platforms, and steps when asking for help.

## Unacceptable behaviour

- Harassment, threats, or personal attacks.
- Discriminatory language or imagery.
- Publishing someone's private information without consent.
- Sustained disruption of discussions or issue threads.
- Encouraging any of the above in others.

## Scope

This applies everywhere Stegcore has an official presence: the GitHub repository, any associated discussions, official social-media accounts, and any private channels used to coordinate the project.

## Enforcement

Report concerns to **ops@themalwarefiles.com**. All reports are handled in confidence. Possible responses include a private warning, a request for a public apology, a temporary ban from project spaces, or a permanent ban. The maintainer reserves the right to remove any content, issue, comment, or participant that violates this code, with no prior notice where safety is at stake.
